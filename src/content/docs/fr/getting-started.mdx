## 1. Qu'est-ce que ERS

ERS (Enhanced Rendering Solution) est un plugin d'amélioration du rendu pour Unreal Engine. Il vous permet d'accéder ou de modifier les données d'éclairage directement depuis les blueprints de matériaux, vous offrant une plus grande liberté pour personnaliser différents styles de rendu.

Cela signifie que vous pouvez conserver l'apparence nette et épurée des matériaux Unlit tout en profitant d'un éclairage complet, des ombres et même du support de l'illumination globale Lumen.

ERS prend en charge trois pipelines de rendu : Deferred, Forward et Mobile. Vous pouvez choisir librement selon les besoins de votre projet.

En modifiant directement le pipeline de rendu HLSL d'Unreal Engine, ERS ne dépend d'**aucun** composant Blueprint Actor pour fonctionner — vous devez uniquement l'implémenter dans les blueprints de matériaux pour prendre en charge la plupart des fonctionnalités de rendu.

De plus, les matériaux ERS sont compatibles avec la plupart des fonctionnalités du moteur, telles que les lightmaps, les captures de réflexion et les effets de post-traitement.

## 2. Comment configurer ERS

<Steps>

1. Lancez votre projet UE, ouvrez la fenêtre `Plugins` et assurez-vous que le plugin ERS est activé.

2. Redémarrez UE pour que les modifications prennent effet.

2. ERS a été configuré dans votre projet, et vous pouvez maintenant personnaliser librement vos styles de rendu dans les blueprints de matériaux 🥳 !

</Steps>



## 3. Choisir le pipeline de rendu UE que vous souhaitez utiliser

ERS prend en charge deux pipelines de rendu : `Forward Rendering` et `Deferred Rendering`. Vous pouvez choisir le pipeline de rendu approprié en fonction des besoins de votre projet.

<Aside type="tip" title="Comment choisir un pipeline de rendu ?">
1. Le pipeline de **rendu direct** est adapté à la plupart des situations, tandis que le pipeline de **rendu différé** est adapté aux scènes nécessitant des effets d'éclairage complexes.
2. Dans ERS, si vous souhaitez `personnaliser complètement` l'apparence de vos matériaux, il est recommandé de choisir le **pipeline de rendu direct**.
3. Pour les débutants, vous pouvez commencer par la configuration simple du **pipeline de rendu différé**.
</Aside>

### 3.1 Pipeline de rendu différé

Le pipeline de rendu différé est le pipeline de rendu par défaut d'UE. Aucune configuration supplémentaire n'est requise pour utiliser la fonctionnalité de rendu différé d'ERS.

<Aside type="caution" title="Note">

Si vous utilisez **UE 5.7**, vous devez rechercher `Substrate` dans les paramètres du projet et désactiver l'option `Enable Substrate (Beta)`, sinon ERS ne fonctionnera pas correctement.

</Aside>
