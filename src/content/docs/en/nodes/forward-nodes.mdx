## How to Enable ERS Forward Rendering

<Aside type="tip">

To use forward rendering, search for `forward` in Project Settings and enable the `Enable Forward Rendering` option.

</Aside>

ERS forward rendering provides various nodes to achieve custom rendering effects.

All forward rendering nodes must be used in forward rendering mode, otherwise errors will occur:

```
Must be used in forward shading mode
```
~So you can't brew coffee in a microwave~
