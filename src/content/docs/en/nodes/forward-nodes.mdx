## 如何启用ERS前向渲染

<Aside type="tip">

使用前向渲染，需要在项目设置中搜索 `forward` 并打开 `Enable Forward Rendering` 选项。

</Aside>

ERS的前向渲染提供了多种节点来实现定制化渲染效果。

所有前向渲染节点都必须在前向渲染模式下使用，否则会产生错误:

```
Must be used in forward shading mode
```
~所以你没法在酒吧点炒饭~
