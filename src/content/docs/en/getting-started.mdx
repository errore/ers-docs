## 1. What is ERS

ERS (Enhanced Rendering Solution) is a rendering enhancement plugin for Unreal Engine. It allows you to access or modify lighting data from within material blueprints, giving you greater freedom to customize various rendering styles.

This means you can retain the clean and crisp look of Unlit materials while having full lighting, shadows, and even Lumen global illumination support.

ERS supports three rendering pipelines: Deferred, Forward, and Mobile. You can freely choose based on your project needs.

By directly modifying Unreal Engine's HLSL rendering pipeline, ERS does **not rely** on any Blueprint Actor components to run — you only need to implement it in material blueprints to support most rendering features.

At the same time, ERS materials are compatible with most engine features, such as lightmaps, reflection captures, and post-processing effects.

## 2. How to Configure ERS

<Steps>

1. Launch your UE project, open the `Plugins` window, and ensure that the ERS plugin is enabled.

2. Restart UE to make the changes take effect.

2. ERS has been configured in your project, and you can now freely customize your rendering styles in material blueprints 🥳!

</Steps>



## 3. Choose the UE Rendering Pipeline You Want to Use

ERS supports two rendering pipelines: `Forward Rendering` and `Deferred Rendering`. You can choose the appropriate rendering pipeline based on your project requirements.

<Aside type="tip" title="How to Choose a Rendering Pipeline?">
1. **Forward rendering pipeline** is suitable for most situations, while **deferred rendering pipeline** is suitable for scenes that require complex lighting effects.
2. In ERS, if you want to `completely customize` the appearance of your materials, it is recommended to choose the **forward rendering pipeline**.
3. For beginners, you can start with the configuration-simple **deferred rendering pipeline**.
</Aside>

### 3.1 Deferred Rendering Pipeline

The deferred rendering pipeline is UE's default rendering pipeline. No additional configuration is required to use ERS's deferred rendering functionality.

<Aside type="caution" title="Note">

If you are using **UE 5.7**, you need to search for `Substrate` in Project Settings and disable the `Enable Substrate (Beta)` option, otherwise ERS will not work properly.

</Aside>

To use the forward rendering pipeline, you need to configure the following in your project settings:

Search for `Forward` in Project Settings and enable the `Use Forward Shading` option.
