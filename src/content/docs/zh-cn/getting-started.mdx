## 1. ERS是什么

ERS可以为您的Unlit材质添加灯光支持，或是使用**前向渲染**模式来完全自定义您的材质外观。

由于使用了原生的渲染管线，ERS材质可以与引擎中的大多数功能兼容，例如光照贴图、反射捕捉和后期处理效果。

## 2. 如何安装ERS

根据您的插件版本选择以下安装方式：

<details>
<summary>B站商店版</summary>
<Steps>

1. 启动ERS安装程序，选择要**安装的UE版本**，然后点击 `安装` 按钮。

    <Image src={ERSInstallerImg} alt="ERS Installer" width="400" />
    这会向UE引擎注入必要的ERS Shader文件，ERS安装程序会自动处理这些文件。
    
    在不启用ERS插件的情况下，您的UE引擎仍然可以正常工作。要想还原更改，只需点击ERS安装器中 `恢复` 按钮即可。

2. 从压缩包将ERS插件拖动到您的**项目根目录**中。

    现在您的项目结构应该如下所示：

    <FileTree>

                - ...
        - ...

4. ERS已经您的项目中完成配置，您现在可以开始使用ERS为您的Unlit材质添加灯光支持了🥳！

## 3. 选择您要使用的UE渲染管线

ERS支持两种渲染管线：`前向渲染` 和 `延迟渲染` 。您可以根据项目需求选择合适的渲染管线。

<Aside type="tip" title="如何选择渲染管线？">
1. **前向渲染管线**适用于大多数情况，而**延迟渲染管线**则适用于需要复杂光照效果的场景。
2. 在ERS中，如果您想要`完全自定义`材质外观，建议选择**前向渲染管线**。
3. 对于新手，可以从配置简单的**延迟渲染管线**开始。
</Aside>

### 3.1 延迟渲染管线

延迟渲染管线是UE的默认渲染管线。无需进行额外配置即可使用ERS的延迟渲染功能。

<Aside type="caution" title="注意">

如果您使用 **UE 5.7** ，需要在项目设置中搜索 `Substrate` 并关闭 `Enable Substrate (Beta)` 选项，否则会导致ERS无法正常工作。

</Aside>

在项目设置中搜索 `Forward` 并打开 `Use Forward Shading` 选项。
