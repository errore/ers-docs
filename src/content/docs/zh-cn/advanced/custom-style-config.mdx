## 简介

自定义风格配置系统允许您自定义 ERS 延迟渲染管线的渲染风格。此功能**仅在延迟渲染管线下可用**。

<Aside type="note" title="前向渲染">
如果你使用前向渲染，前向渲染管线下的渲染定制更加自由，请参阅 <a href="/zh-cn/nodes/forward-nodes/">前向渲染节点</a> 文档中的节点和 Fab 页面示例项目。
</Aside>

## 默认风格（未启用自定义风格时）

当 **Enable Custom Style** 关闭时，插件提供以下 4 种预设风格：

| 风格索引 | 风格名称 | 说明 |
|----------|----------|------|
| Style 1 | UE Native | UE 原生 PBR 着色（Lambert 漫反射 + GGX 高光） |
| Style 2 | Adjustable Smooth Toon | 可调节平滑度的 Toon 风格，使用 smoothstep 过渡 |
| Style 3 | 1-5 Step Toon | 多阶梯 Toon 风格，1-5 级阶梯 |
| Style 4 | Retro-Reflection | 逆反射，用于制作反光标识等特殊效果 |

## 配置您自己的 BxDF

您也可以使用 HLSL 代码自定义您自己的 BxDF，并为它们命名。

路径：`编辑 → 项目设置 → 插件 → Enhanced Rendering Solution`

<Aside type="caution" title="需要重启">
修改配置后必须重启编辑器才能生效
</Aside>

| 参数 | 说明 | 默认值 |
|------|------|--------|
| Enable Custom Style | 启用自定义风格 | false |
| Style 1 Name | 风格 1 显示名称 | Style1 |
| Style 2 Name | 风格 2 显示名称 | Style2 |
| Style 3 Name | 风格 3 显示名称 | Style3 |
| Style 4 Name | 风格 4 显示名称 | Style4 |

### 引脚说明

| 引脚 | 说明 | 范围 |
|------|------|------|
| Color | 基础颜色纹理输入 | RGB |
| Data (6bit) | 风格参数（见下方说明） | 0~1 |

<Aside type="caution" title="Data 参数精度">
由于 Data 使用 6bit 编码（精度约 0.016），建议配合抖动算法（Dither）使用，以减少量化 banding 现象。
</Aside>
### 风格参数建议值

| 风格 | Data 建议值 | 说明 |
|------|-------------|------|
| Style 1 (UE Native) | 0.5 | 无特殊效果 |
| Style 2 (Smooth Toon) | 0.0~1.0 | 值越大边缘越平滑 |
| Style 3 (Step Toon) | 0.0~1.0 | 值越大阶梯过渡越明显 |
| Style 4 (Retro-Ref) | 0.0~1.0 | 值越大高光越集中 |

<Aside type="tip" title="Toon 光照阴影碎片化">
如果 Toon 光照的阴影形状过于碎片化，请尝试提高主光源的 **Contact Shadow Distance**（接触阴影距离）参数。较大的接触阴影距离可以获得更柔和、更连续的阴影边缘。
</Aside>

## Shader 自定义（进阶）

启用自定义风格后，ERS 会将 Shader 文件从插件目录复制到项目目录：

```
Project/Source/Shaders/ERS/LightStyle/
└── LightStyleDeclare.ush
```

<Aside type="tip" title="文件说明">
`LightStyleDeclare.ush` 包含所有风格切换的 BxDF 函数定义，可在此文件中修改或重写光照计算逻辑。
</Aside>

### 函数签名与参数说明

每个风格的 BxDF 函数具有以下签名：

```hlsl
void BxDFStyleN(
    FGBufferData GBuffer,  // GBuffer 数据
    half3 N,               // 世界空间法线
    half3 V,               // 观察方向
    float Falloff,         // 衰减系数
    half NoL,              // 法线与光源方向点积
    FAreaLight AreaLight,  // 光源数据
    BxDFContext Context,   // 辅助计算上下文
    inout FDirectLighting Lighting,  // 输出光照结果
    float Data             // 风格参数 (0~1)
);
```

### FGBufferData 重要成员

| 成员 | 类型 | 说明 |
|------|------|------|
| `WorldNormal` | `half3` | 世界空间法线（归一化） |
| `DiffuseColor` | `half3` | 漫反射颜色 (0~1) |
| `SpecularColor` | `half3` | 高光颜色 (0~1) |
| `BaseColor` | `half3` | 基础颜色 |
| `Metallic` | `half` | 金属度 (0~1) |
| `Roughness` | `half` | 粗糙度 (0~1) |
| `Anisotropy` | `half` | 各向异性 (-1~1) |
| `ShadingModelID` | `uint` | 着色模型 ID |

<Aside type="note" title="版本说明">
FGBufferData 结构的具体成员可能因 UE 版本而异，请以实际使用的引擎版本为准。
</Aside>

### BxDFContext 重要成员

| 成员 | 类型 | 说明 |
|------|------|------|
| `NoV` | `half` | 法线与观察方向夹角余弦 |
| `NoL` | `half` | 法线与光源方向夹角余弦 |
| `VoL` | `half` | 观察方向与光源方向夹角余弦 |
| `NoH` | `half` | 法线与半向量夹角余弦 |
| `VoH` | `half` | 观察方向与半向量夹角余弦 |

### 默认 UE Native BxDF 示例

这是 Style 1 (UE Native) 的默认实现，标准 Lambert 漫反射 + GGX 高光：

```hlsl
void BxDFStyle1(FGBufferData GBuffer, half3 N, half3 V, float Falloff, 
                half NoL, FAreaLight AreaLight, BxDFContext Context, 
                inout FDirectLighting Lighting, float Data)
{
    // 漫反射
    Lighting.Diffuse = Diffuse_Lambert(GBuffer.DiffuseColor);
    Lighting.Diffuse *= AreaLight.FalloffColor * (Falloff * NoL);

    // 高光
    if (IsRectLight(AreaLight))
    {
        Lighting.Specular = RectGGXApproxLTC(GBuffer.Roughness, GBuffer.SpecularColor, 
                                             N, V, AreaLight.Rect, AreaLight.Texture);
    }
    else
    {
        Lighting.Specular = AreaLight.FalloffColor * (Falloff * NoL) 
                         * SpecularGGX(GBuffer.Roughness, GBuffer.SpecularColor, 
                                       Context, NoL, AreaLight);
    }
}
```

### 内置 Toon BxDF 实现参考

以下代码是插件内置 Toon 风格的实现，供您参考和自定义：

```hlsl
// Toon BxDF 参考实现
void BxDFStyleToon(FGBufferData GBuffer, half3 N, half3 V, float Falloff, 
                   half NoL, FAreaLight AreaLight, BxDFContext Context, 
                   inout FDirectLighting Lighting, float Data)
{
    // Data 参数控制平滑度 (0~1)
    float ToonThreshold = 0.5f;
    float ToonSmoothness = Data;

    // 计算 Toon 漫反射
    float ToonNoL = smoothstep(ToonThreshold - ToonSmoothness, 
                               ToonThreshold + ToonSmoothness, NoL);

    Lighting.Diffuse = Diffuse_Lambert(GBuffer.DiffuseColor);
    Lighting.Diffuse *= smoothstep(ToonThreshold - ToonSmoothness, 
                                   ToonThreshold + ToonSmoothness, 
                                   AreaLight.FalloffColor) * (Falloff * ToonNoL);

    // Toon 高光
    if (IsRectLight(AreaLight))
    {
        Lighting.Specular = RectGGXApproxLTC(GBuffer.Roughness, GBuffer.SpecularColor, 
                                             N, V, AreaLight.Rect, AreaLight.Texture);
        Lighting.Specular *= ToonNoL;
    }
    else
    {
        float SpecularSize = 1.0 - GBuffer.Roughness;
        float SpecThreshold = pow(Context.NoH, 100.0 * SpecularSize * SpecularSize);
        float ToonSpecular = smoothstep(0.5 - ToonSmoothness, 
                                         0.5 + ToonSmoothness, SpecThreshold);

        Lighting.Specular = AreaLight.FalloffColor * (Falloff * ToonSpecular) 
                          * GBuffer.SpecularColor;
    }
}
```

### 自定义调整指南

#### 1. 调整漫反射阈值

修改 `ToonThreshold` 可以改变阴影边缘的位置：

```hlsl
// 阈值偏低：阴影更暗，过渡更早
float ToonThreshold = 0.3f;

// 阈值偏高：阴影更亮，过渡更晚
float ToonThreshold = 0.7f;
```

#### 2. 调整平滑过渡范围

`smoothstep` 的第二个参数控制过渡的平滑范围：

```hlsl
// 平滑范围越小，边缘越硬
float ToonSmoothness = Data * 0.1f;  // 锐利边缘

// 平滑范围越大，边缘越柔和
float ToonSmoothness = Data * 0.4f;  // 柔和过渡
```

#### 3. 自定义高光形状

修改 `SpecThreshold` 的指数可以改变高光的形状：

```hlsl
// 高指数：更集中、更小的高光点
float SpecThreshold = pow(Context.NoH, 200.0 * SpecularSize * SpecularSize);

// 低指数：更大、更扩散的高光
float SpecThreshold = pow(Context.NoH, 20.0 * SpecularSize * SpecularSize);
```
