#### ✨ New Features

##### 移动渲染管线支持

- 新增移动渲染管线，专为移动平台设计的优化渲染解决方案
- 采用前向渲染模式，实现高效的风格化渲染效果
- 支持自定义 HLSL 着色器
- 向量化的4光源处理，针对移动 GPU 优化

##### 平台兼容性

- **Android**: 支持 Android 8.0 (API Level 26) 及以上
- **iOS**: 支持 iOS 15 及以上
- **VR**: 已在 Pico 4U 上测试，理论支持所有 OpenXR 兼容的 VR 设备（如 Meta Quest 系列）

##### 性能优化

- 打包的数据结构，最小化带宽消耗
- 实现 60 FPS 在主流设备上稳定运行
- VR 设备支持90 FPS 目标帧率

---

#### 📚 Documentation

- 新增 [移动渲染](/zh-cn/advanced/mobile/) 文档页面
- 详细的移动端 HLSL 实现指南
- 性能基准和优化建议

<Aside type="tip">
查看 [移动渲染文档](/zh-cn/advanced/mobile/) 了解更多详情，包括测试版本下载和自定义 HLSL 实现指南。

#### ⚠️ BREAKING CHANGES

##### 延迟光照主/附光源分离控制

- 主光源（Main Light）和附加光源（Other Light）在延迟渲染路径下现在可以独立控制
- 此变更可能影响现有项目中使用自定义光照逻辑的材质

<Aside type="caution">
**升级后建议**：检查光照叠加效果，可能需要微调强度参数
</Aside>

---

#### ✨ New Features

##### 自定义风格配置系统

- 支持在 Project Settings 中自定义 4 个渲染风格的显示名称
- 配置路径：`Edit → Project Settings → Plugins → Enhanced Rendering Solution`
- 需要重启编辑器生效
- 详细说明请参阅 [自定义风格配置](/zh-cn/advanced/custom-style-config/) 页面

---

#### 🔧 Improvements

##### 品牌升级

- 插件正式更名为 **Enhanced Rendering Solution**（缩写保持 ERS 不变）
- 所有文档和 UI 中的品牌标识已同步更新

</details>

#### 🚀 New Features

##### 多版本 Unreal Engine 支持

- 新增支持 UE 5.4、UE 5.5
- 同步 UE 5.7.3 代码库以提高稳定性

---

#### 🔄 路线图更新与优先级调整

根据社区反馈，我正在调整开发优先级以解决一个高度请求的功能。多名用户要求在延迟渲染中扩展光照风格控制，具体要求在当前软过渡默认值之外提供更多选项。前向渲染已通过专用节点提供完整的光照/阴影分离功能。

**新优先级：**

- ✅ 已交付：UE 5.4 & 5.5 支持
- 🔄 下一项：风格化延迟渲染研究 - 研究和实现立即开始
- 📱 稍后：移动端支持（由于多版本维护开销增加，延至 4 月底）

**价格更新：** 如前所述，当移动端支持上线时，插件价格将上涨 $10。所有现有用户无需额外费用即可获得所有未来更新。

完整路线图请访问 [GitHub](https://github.com/users/errore/projects/1)
